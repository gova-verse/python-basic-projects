# Calculator (Python)

A beginner-friendly command-line calculator built in Python.

## Features
- Basic arithmetic operations:
  - Addition (`+`)
  - Subtraction (`-`)
  - Multiplication (`*`)
  - Division (`/`)
- Takes user input from the terminal
- Prints the calculated result instantly

## Project Goal
This project helps beginners practice:
- `input()` handling
- Type conversion (`int`/`float`)
- Conditional logic (`if`/`elif`)
- Defining and calling functions

## File Structure
```text
01_calculator/
├── calculator.py
└── README.md
```

## How to Run
From the repository root:

```bash
python 01_calculator/calculator.py
```

## Example
```text
Enter first number: 10
Enter operator (+, -, *, /): *
Enter second number: 5
Result: 50
```

## Notes
- Handle division by zero gracefully in your calculator logic.
- You can extend this project by adding loops and repeated calculations.
